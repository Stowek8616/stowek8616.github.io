What does this website do?

This website helps you track your progress with weapon overclocks in Deep Rock Galactic. Use the colored class links above to navigate to each class's page, where you can check off which overclocks you've acquired and monitor your collection progress. Checked off boxes will be saved between multiple browsing sessions.

Each page is sorted by the weapons available to each class and then sorted by three Overclock types. Clean, Balanced, and Unstable, which indicated how much it changes the weapon. At the bottom of each page is a link to my email and the Github Repo that hosts this site.
